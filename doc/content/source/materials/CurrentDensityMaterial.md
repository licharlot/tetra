# CurrentDensityMaterial

## Description

This material calculates the current density in presence of the Seebeck effect:
\begin{equation}
 \vec{J} = \frac{1}{\rho_e} (\nabla V + S \nabla T)
\end{equation}

where $V$ is the electric potential, $\vec{J}$ is the current density.
